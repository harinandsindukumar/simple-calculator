# Contributing to Animated Calculator

Thank you for considering contributing to the Animated Simple Calculator project! We welcome contributions of all types.

## Getting Started

1. Fork the repository.
2. Clone the repository:
   ```bash
   git clone https://github.com/harinandsindukumar/simple-calculator.git
   ```
3. Navigate to your local repository:
   ```bash
   cd simple-calculator
   ```

## How to Contribute

### Report Issues
If you find a bug, please open an issue and describe both the problem and the solution you're considering.

### Code Contribution

1. Create a new branch:
   ```bash
   git checkout -b feature-your-feature
   ```
2. Make changes and commit:
   ```bash
   git commit -m 'Describe your changes briefly but clearly'
   ```
3. Push your changes:
   ```bash
   git push origin feature-your-feature
   ```
4. Open a Pull Request.
